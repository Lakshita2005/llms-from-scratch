# LangChain Models Practice

This repository contains hands-on implementations of:

- LLMs
- Chat models (Open-source & Closed-source)
- Embedding models
- Document similarity using cosine similarity

## Folder Structure

- 1.LLMs/ – Basic LLM usage
- 2.ChatModels/ – Chat models using API and local models
- 3.EmbeddedModels/ – Embeddings, document similarity, vector search

## Tech Stack

- Python
- LangChain
- Hugging Face
- OpenAI
- Scikit-learn
